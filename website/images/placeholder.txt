## Image Guidelines:

1. Use JPEG for photographs, PNG for logos/graphics
2. Optimize images for web (compress to reduce file size)
3. Recommended tools: TinyPNG, Squoosh, ImageOptim
4. Keep file sizes under 200KB for fast loading
5. Use descriptive file names for SEO

Consider using stock photos or illustrations instead.
